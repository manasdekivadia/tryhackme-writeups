
# TryHackMe Writeups

A curated collection of my writeups from TryHackMe, documenting my hands-on journey in cybersecurity.
This repository showcases practical skills in penetration testing, network security, web exploitation, and privilege escalation.

## About This Repository

This repo contains:

* Structured writeups of TryHackMe rooms
* Step-by-step methodologies (without violating platform rules)
* Key concepts and learnings from each lab
* Tools and techniques used during solving

Note:
This repository does not include flags, passwords, or direct answers, in accordance with TryHackMe’s content policy.
